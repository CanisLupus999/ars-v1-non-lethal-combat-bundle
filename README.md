# [ARS] V1 Non-lethal Combat Bundle

A Foundry VTT content module for the ARS system providing non-lethal combat options including macros, items, and documentation.

## Installation

1. In Foundry VTT, go to the **Add-on Modules** tab
2. Click **Install Module**
3. Paste the manifest URL (to be added after GitHub setup)
4. Enable the module in your world's module settings

## Contents

- **Macros**: Non-lethal combat macros
- **Items**: Non-lethal combat items
- **Documentation**: Usage instructions and guides

## Compatibility

- **Minimum Foundry Version**: 13
- **Verified Foundry Version**: 13
- **Required System**: ARS
